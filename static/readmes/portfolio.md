# Portfolio Website

A modern, interactive portfolio website built with Svelte and TailwindCSS.

## Features

- Responsive design
- Dark/Light mode
- Project showcase
- Skills section
- Contact form
- Blog integration
- SEO optimized
- Performance optimized
- Analytics integration

## Tech Stack

- Svelte/SvelteKit
- TypeScript
- TailwindCSS
- Markdown support
- Contact form with EmailJS
- Google Analytics
- Vercel deployment

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
